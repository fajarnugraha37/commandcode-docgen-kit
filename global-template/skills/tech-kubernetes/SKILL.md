# Kubernetes Discovery Heuristics

Inspect Deployments/StatefulSets/Jobs/CronJobs, Services, Ingress/Gateway, ConfigMaps, Secret references, probes, resources, autoscaling, affinity, disruption budgets, service accounts, and network policies. Manifests describe intended deployment; do not claim live cluster state unless runtime evidence is provided.
