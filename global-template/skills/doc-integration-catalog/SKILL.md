# Integration catalog

Inventory every evidenced dependency outside the current component/process boundary:

- internal downstream/upstream services;
- third-party APIs;
- cloud-managed services;
- object storage;
- databases/search/cache;
- message brokers/streams;
- identity/security services;
- workflow engines;
- file/SFTP integrations;
- observability and operational services.

Capture direction, protocol/client, configuration source, authentication, data exchanged, call sites, timeout/retry/failure behavior, and criticality when evidenced.
