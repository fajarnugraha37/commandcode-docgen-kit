# Ownership Analysis

Model explicit teams and responsibilities, RACI assignments, component owners, data owners, operational/runbook/on-call owners, approval authority, and escalation paths. Missing ownership must remain UNKNOWN; never guess from commit authors.

## Evidence boundary

Use only paths listed in `.docgen/state/source-files.txt` or existing `.docgen` model/evidence artifacts. All FACT items require resolvable non-ignored source evidence.
