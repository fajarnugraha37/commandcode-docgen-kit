# Semantic Quality

Prioritize evidence density and semantic completeness over prose length. Check claim grounding, declared evidence usage, catalog item coverage, decision/branch coverage, stale fingerprints, unsupported claims, cross-page contradictions, and unintentional duplication. Word count is advisory only.
