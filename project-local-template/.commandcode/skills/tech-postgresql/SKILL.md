# PostgreSQL Discovery Heuristics

Inspect migrations/DDL, schema-qualified objects, constraints, indexes, views, materialized views, functions, procedures, triggers, sequences/identity, LISTEN/NOTIFY, and SQL embedded in mappers/DAOs. Distinguish declared schema from actual deployed state unless deployment evidence exists.
