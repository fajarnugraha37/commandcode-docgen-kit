# MyBatis Discovery Heuristics

Inspect `mybatis-config.xml` or programmatic `Configuration`, mapper registration, mapper interfaces, XML namespaces, statement ids, `resultMap`, `sql` fragments, dynamic SQL, type handlers, environments, and transaction factory/data source integration. Map interface methods to XML statements only when namespace and ids support the relationship.
