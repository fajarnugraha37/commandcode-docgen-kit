# Kafka Discovery Heuristics

Identify producers/consumers, topic sources, key strategy, serializers/deserializers, consumer groups, offset/ack behavior, retries, dead-letter topics, transactions/idempotence, headers/correlation, Kafka Streams topology, and schema contracts. Treat exactly-once or ordering guarantees as unknown unless configuration and processing boundaries support them.
