# JPA / EclipseLink Discovery Heuristics

Inspect `persistence.xml`, provider configuration, entities, embeddables, relationships, named queries, converters, listeners, repositories/DAOs, and transaction boundaries. Do not confuse object relationships with database ownership or cascade behavior unless mappings explicitly support it.
