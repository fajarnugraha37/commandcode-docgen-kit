# HK2 Discovery Heuristics

Inspect `AbstractBinder`, `bind`, `bindAsContract`, factories, scopes, qualifiers, and Jersey registration. Distinguish service availability from actual injection use. Record lifecycle/scope only where explicit or reliably framework-defined for the exact binding.
